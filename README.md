# Sushant Kumar – Developer Portfolio

A modern and animated personal portfolio website built using HTML, CSS, and JavaScript. This project showcases my skills, projects, education, and contact information through a clean and interactive interface.

---

## Live Demo

The portfolio can be deployed using platforms like Vercel or GitHub Pages.

Example:

https://your-portfolio.vercel.app

---

## About the Project

This portfolio website was created to demonstrate my frontend development skills and to provide a professional platform where visitors can learn about my technical abilities, projects, and academic background.

The website features smooth animations, interactive UI elements, and a responsive design to ensure a great experience across desktop and mobile devices.

---

## Features

* Animated particle background
* Typing animation in the hero section
* Glass-style modern UI design
* Scroll reveal animations
* Interactive skill cards
* Project showcase section
* Smooth scrolling navigation
* Fully responsive layout for mobile and desktop

---

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Particles.js
* ScrollReveal.js

## Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/portfolio.git

Open the folder in your code editor and run the project by opening the `index.html` file in your browser.

Since this is a static website, no additional setup or dependencies are required.
